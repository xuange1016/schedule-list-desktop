# 项目文档索引

本目录保存桌面端任务管理应用 V1 的产品与设计说明。后续切换对话窗口或重新开始开发时，AI 应优先阅读这些文档。

## 推荐阅读顺序

1. `todoist-like-desktop-app-prd-v1.md`
   - 产品需求说明书
   - 说明目标用户、页面需求、功能范围、验收标准

2. `todoist-like-desktop-app-wireframes-v1.md`
   - 页面原型与低保真布局说明
   - 说明每个页面大概怎么摆、弹窗长什么样、按钮在哪

3. `todoist-like-desktop-app-user-flows-v1.md`
   - 用户流程与任务状态规则
   - 说明任务如何在收件箱、今天、预览、过滤、日志之间流转

4. `todoist-like-desktop-app-implementation-plan-v1.md`
   - 开发实施计划
   - 说明 V1 应按哪些里程碑推进、每阶段交付什么、如何验收

## 当前产品决策摘要

- 暂时只做桌面端
- 第一版只做本地保存
- 界面语言使用中文
- 整体风格参考 Todoist，但不复制品牌资源
- 任务详情使用弹窗编辑
- 附件第一版需要真正上传文件
- 日志任务支持恢复为未完成
- 预览页面默认展示未来 30 天，并且每天都显示
- 标签默认包含学习、生活、工作，同时支持用户自定义
- 删除任务需要双重确认，删除后不可撤回
- 收件箱是所有未完成任务的总入口
