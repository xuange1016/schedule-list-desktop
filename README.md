# 日程清单

中文桌面端任务管理应用，当前处于 V1 的 M8 阶段：视觉打磨、验收和打包准备。

## 当前状态

已完成：

- Electron 桌面应用外壳
- React + TypeScript 前端入口
- 左侧导航栏
- 收件箱、今天、预览、过滤、日志五个页面空壳
- 中文界面基础样式
- 任务数据模型
- Electron 本地任务存储
- 应用用户数据目录下的 `data/tasks.json` 持久化文件
- 前端读取本地任务
- 基础添加任务验证入口
- 任务完成与日志恢复的基础状态切换
- 预览页面未来 30 天日期分组
- 过滤页面按标签和优先级筛选
- 新建任务弹窗
- 编辑任务弹窗
- 标题、描述、日期、优先级、标签、提醒字段
- 标题为空校验
- 保存和取消
- 删除任务双重确认
- 单任务永久删除
- 收件箱按今天、未来安排、无日期、逾期分组
- 今天页面只显示今日任务
- 预览页面只显示未来 30 天内有日期的未完成任务
- 无日期任务只显示在收件箱和过滤结果中
- 逾期任务保留在收件箱中
- 日期状态标签：今天、明天、无日期、逾期
- 完成任务后显示轻量反馈
- 日志按完成日期分组
- 日志显示完成时间、原截止日期、标签、优先级和描述摘要
- 日志任务支持恢复为未完成
- 恢复后任务回到对应未完成页面
- 过滤页面按优先级筛选
- 过滤页面按标签筛选
- 默认标签：学习、生活、工作
- 自定义标签从任务中自动出现
- 筛选结果计数
- 当前筛选摘要
- 清除筛选
- 空结果状态
- 任务弹窗支持选择并上传附件
- 附件会复制到应用本地附件目录
- 任务中保存附件记录
- 弹窗中显示附件文件名和大小
- 任务行显示附件数量
- 可以从任务弹窗打开已上传附件
- 同名附件不会互相覆盖
- 统一按钮、焦点、滚动条和任务行交互状态
- 基础视觉打磨和可访问性焦点状态
- Electron Builder 打包配置已写入 `package.json`

暂未完成：

- 生成正式安装包前需要安装 `electron-builder`

## 启动方式

安装依赖：

```bash
npm install
```

启动开发环境：

```bash
npm run dev
```

类型检查：

```bash
npm run typecheck
```

构建前端：

```bash
npm run build
```

打包准备：

```bash
npm install --save-dev electron-builder
npm run dist
```

如果只想生成未安装的应用目录：

```bash
npm run pack
```
